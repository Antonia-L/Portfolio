Road Trip Story

This is a project created by my partner, Brandon Lau, and I using Scratch in Computing Science/Engineering class. We worked on the project between August 29, 2017 and September 8, 2017. We created a story for this project. The objective of the story is to help Bear, the main character in the story, to make decisions in his road trip. I created some of the plots in the story, and I coded for most of the story. When I was doing the project, I learned how to code to make the sprite does certain things. I also learned how to use variables. One of the challenges I had when coding for the project is the events goes in loops. I solved the problem by using different ways to code that will reach the same result.
