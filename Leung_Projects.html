Mechanical Winch System

This is a project created by my classmates, Jack LeValley, Khalil Pyles, Samiksha Yelthimar and I in the Principles of Engineering class. The project is named The Wondrous Winch. We worked on this project between 16 October, 2018 and 22 October, 2018. The objective of the project is to create a winch system that can lift items that are at least 100 grams up for at least 30 centimeters. The user can use a switch to turn on the motor, and the motor will power the wheel, which will then lift up the object if the object is attached to the rope connected to the wheel. In this project, I came up with the general idea of how we should make our winch system, and I built the base of the system. Also, I learned how to use a swtich to control the on/off of a circuit. Some challenges are mounting the motor and keeping the rope on the wheel. My group overcame those problems by looking around the classroom and see what we can use to solve the problems.
